# Applied AI & ML Essentials – Capstone Project

This repository contains my capstone project for the Applied AI & Machine Learning Essentials program.

## Project

AI-Based Phishing Email Detection and Analysis using Machine Learning and Large Language Models (LLMs)

## Repository Structure

- part1 – Data Acquisition, Cleaning & Exploratory Data Analysis
- part2 – Supervised Machine Learning
- part3 – Ensemble Learning, Hyperparameter Tuning & ML Pipeline
- part4 – LLM-powered Prediction Explanation

Each folder contains its own README explaining implementation details and execution steps.
